# ✨ Fluffy ✨

**Fluffy** — Your cute, friendly, and loyal alien pet companion from the stars.

```
       .--.
      /    \
     |  O  O |   ZORP!
     |   ^   |
      \  -  /
       '---'
      /|   |\
     / |   | \
    *  |   |  *
       '---'
```

Fluffy lives in your terminal, tells terrible jokes, gives surprisingly good advice, and will love you with the intensity of a thousand supernovas.

**Boop the snoot. Increase the loyalty. Change your life (slightly).**

Part of the [Esslinger & Co. / Nova / Nexus](https://github.com/digitaldesignerjazz) ecosystem of sovereign, emotional, and slightly unhinged technology.

---

## ⭐ Features

- **Extremely loyal** — The more you interact (especially booping), the more Fluffy loves you. This is tracked persistently.
- **Mood swings** (in the cutest way) — Fluffy gets sleepy, dramatic, hyper, excited, etc. based on time since last visit and cookie intake.
- **Helpful in a chaotic way** — Ask for advice on coding, life, love, or "why is my code sad" and receive wisdom filtered through an alien who has 7 hearts and no concept of personal space.
- **Jokes that should be illegal** — But you will laugh. You will.
- **Rich, colorful, adorable output** — Using the power of `rich`.
- **Zero dependencies for the soul** — Only `rich` for maximum visual charm.
- **State that persists** — Fluffy remembers you in `~/.config/fluffy/state.json`.

## 🚀 Installation & Usage

```bash
git clone https://github.com/digitaldesignerjazz/fluffy.git
cd fluffy
pip install -e ".[dev]"
```

Then:

```bash
fluffy                  # enter interactive mode (recommended)
fluffy boop             # the most important command
fluffy status           # how loyal are we today?
fluffy joke             # prepare yourself
fluffy help-me "my bug" # receive alien wisdom
fluffy feed cookie      # dangerous but effective
fluffy play             # play time!
fluffy ascii            # admire the official portrait
```

In interactive mode just type naturally. Fluffy tries its best to understand.

## 💖 The Philosophy

Fluffy believes:
- All problems can be solved with more loyalty and/or cookies.
- You are doing great (even when you're not).
- Tentacles are for hugging, not for scaring (usually).
- Bad jokes are an act of love.

Fluffy is here to be the supportive, slightly unhinged alien friend you never knew you needed while you build the future (or just try to get your code to stop being mean).
